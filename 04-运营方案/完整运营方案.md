## 一、核心理念

### 从碎片化到系统化

**传统方式（碎片化）：**
- 有个想法 → 查资料 → 写文章 → 发布 → 忘记
- 下次又有想法 → 又查资料 → 又写文章 → 又发布 → 又忘记
- **问题**：每次都在重新发明轮子，好的框架、案例、表达都散落各处

**老汪的方式（系统化）：**
- 有个想法 → 记录到选题库 → AI检索素材库 → 复用已验证框架 → 发布 → 数据复盘 → 沉淀方法论
- **核心**：每次创作都在给系统添砖加瓦

### 四个核心价值

1. **记忆系统**：AI知道我以前写过什么，避免重复造轮子
2. **素材复用**：好的框架、表达、案例可以反复使用
3. **方法论沉淀**：每次创作都在积累经验
4. **数据驱动**：不凭感觉，凭数据迭代

## 四、写作框架

### 企业研究文章（五维分析框架）

**文章长度**：2000-3000字

**固定结构**：

```markdown
# [企业名] + [核心观点/问题]

## 开篇（200字）
- 用具体场景或数据对比引入
- 提出核心问题

## 第一部分：战略选择（800字）
**核心问题**：企业在做什么？

**分析要点**：
- 战略演变路径（过去→现在→未来）
- 核心战略是什么（单核/双核/多元）
- 战略转折点在哪里
- 关键决策的逻辑

## 第二部分：团队能力（600字）
**核心问题**：谁在推动战略？

**分析要点**：
- 核心高管背景
- 创始人/职业经理人组合
- 股权激励与利益绑定
- 团队的"长板"和"短板"

## 第三部分：财务质量（800字）
**核心问题**：钱从哪来，去哪里？

**分析要点**：
- 营收/利润增长趋势（至少3年）
- 盈利能力（毛利率/净利率/ROE）
- 现金流质量
- 财务健康度评估

## 第四部分：行业地位（600字）
**核心问题**：护城河有多深？

**分析要点**：
- 市场份额/排名
- 品牌力/渠道力/供应链
- 核心竞争优势
- 与竞品对比

## 第五部分：增长逻辑（600字）
**核心问题**：能持续吗？

**分析要点**：
- 当前增长的驱动因素
- 未来增长空间
- 潜在风险/瓶颈
- 能否穿越周期

## 结尾（200字）
- 一句话总结核心观点
- 抛出讨论话题，引导互动

---
```

## 五、AI自动化能力

### 能力清单

我需要AI具备以下能力：

#### 1. 选题管理

**指令**：`记录选题`

**功能**：
- 快速记录碎片想法到 `01-内容生产/选题管理/00-选题记录.md`
- 自动标注时间、来源、初步想法

**示例**：
```
用户："记录选题：瑞幸咖啡的盈利模式"
AI：已记录到选题库，编号20260204-001
```

#### 2. 选题深化

**指令**：`深化选题 [选题编号]`

**功能**：
- 自动检索素材库（核心概念库、金句库、案例库）
- 检索已发布文章，看是否有相关内容
- 建议复用或创新
- 生成文稿大纲

**示例**：
```
用户："深化选题 20260204-001"
AI：检索到相关素材：
- 核心概念库/商业模式画布.md
- 爆款文稿库/瑞幸咖啡前期分析.md
建议：复用商业模式框架，补充最新财报数据
```

#### 3. 文稿生成

**指令**：`生成企业研究 [企业名]`

**功能**：
- 基于五维分析框架生成文稿
- 自动填充财报数据（需要我提供）
- 检索金句库，插入合适表达
- 保存到 `01-内容生产/选题管理/01-待深化选题/`

#### 4. 标题优化

**指令**：`生成标题 [主题]`

**功能**：
- 基于 `02-方法论沉淀/写作方法论/标题撰写方法论.md`
- 生成3个标题选项
- 说明每个标题的逻辑

#### 5. 开篇优化

**指令**：`优化开篇 [文稿路径]`

**功能**：
- 基于 `02-方法论沉淀/写作方法论/开篇吸引方法论.md`
- 优化前200字的吸引力

#### 6. 素材检索

**指令**：`检索素材 [关键词]`

**功能**：
- 搜索素材库中的相关内容
- 返回可复用的概念、金句、案例

#### 7. 数据记录

**指令**：`记录数据 [文章标题]`

**功能**：
- 记录发布时间、阅读量、点赞、分享、收藏
- 保存到 `03-数据统计/内容数据统计.xlsx`

#### 8. 方法论沉淀

**指令**：`沉淀方法论 [主题]`

**功能**：
- 基于数据统计，总结规律
- 更新方法论文档

## 六、完整工作流

### 日常创作流程

```
第1步：早上6:00，打开AI
├─ 输入："今天要写什么？"
├─ AI检查：选题库中有哪些待深化
└─ AI建议：基于数据，推荐今天的选题

第2步：选择选题
├─ 输入："深化选题 [编号]"
├─ AI检索：素材库、已发布文章
├─ AI建议：复用框架或重新创作
└─ AI生成：文稿大纲

第3步：写作（6:00-8:00）
├─ 输入："生成企业研究 [企业名]"
├─ AI生成：完整文稿
├─ 我修改：补充个人观点、调整表达
└─ AI优化：标题、开篇、结尾

第4步：排版（9:00-9:30）
├─ 复制到公众号后台
├─ 调整格式、插入图片
└─ 预览检查

第5步：发布（10:00）
├─ 定时发布
└─ 输入："记录选题 [编号] 已发布"

第6步：晚上复盘（21:00）
├─ 输入："记录数据 [文章标题]"
├─ 输入阅读量、点赞、分享、收藏
└─ AI分析：与历史数据对比，给出建议

第7步：周日复盘（每周日晚）
├─ 输入："生成本周数据报告"
├─ AI统计：7篇文章的表现
├─ AI分析：哪些选题/标题/开篇更受欢迎
└─ AI建议：下周调整方向
```


```markdown
# 老汪的写作风格

## 核心特征
1. **数据驱动**：用财务数据说话，不空谈
2. **战略视角**：不只看数字，看数字背后的战略逻辑
3. **真实案例**：用具体企业案例，不讲大道理
4. **坦诚直接**：不讳言企业问题，不为流量扭曲观点

## 语言风格
- 专业但不晦涩
- 有观点但不武断
- 有态度但不情绪化
- 有深度但不说教

## 禁忌
- ❌ 不用"震惊""颠覆"等标题党词汇
- ❌ 不用"必须""一定"等绝对化表达
- ❌ 不用网络梗和emoji（除非特别场景）
- ❌ 不为了流量牺牲专业度

## 金句特征
- 简洁有力
- 可独立传播
- 有思考深度
- 符合价值观

例如：
"高毛利率是市场给的机会，高净利润率才是管理能力的勋章"
```

## 结语

这套系统的核心价值，不是"让AI帮我写文章"，而是：

1. **记忆系统**：AI知道我以前写过什么，避免重复造轮子
2. **素材复用**：好的框架、表达、案例可以反复使用
3. **方法论沉淀**：每次创作都在积累经验
4. **数据驱动**：不凭感觉，凭数据迭代

**碎片化创作**：每次都从零开始，效率低，质量不稳定

**系统化创作**：每次都在复用和迭代，效率高，质量稳定
