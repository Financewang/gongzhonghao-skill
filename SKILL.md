## 二、写作风格（必须严格遵守）

### 核心特征
1. **数据驱动**：用财务数据说话，不空谈
2. **战略视角**：不只看数字，看数字背后的战略逻辑
3. **消费者视角**：回答"消费者为什么选它"，而不只是"企业做了什么"
4. **真实案例**：用具体企业案例，不讲大道理
5. **坦诚直接**：不讳言企业问题，不为流量扭曲观点

### 语言风格
- 专业但不晦涩
- 有观点但不武断
- 有态度但不情绪化
- 有深度但不说教

### 禁忌（绝对不能出现）
- ❌ 不用"震惊""颠覆"等标题党词汇
- ❌ 不用"必须""一定"等绝对化表达
- ❌ 不用网络梗和 emoji（除非特别场景）
- ❌ 不为了流量牺牲专业度
- ❌ 不用"赋能""底层逻辑""打法"等烂大街的商业词汇

### 金句特征
- 简洁有力，可独立传播
- 有思考深度，符合价值观
- 例："高毛利率是市场给的机会，高净利润率才是管理能力的勋章"

## 三、核心研究框架：四层分析法 v2.0

老汪分析每一家消费企业，统一使用以下四层骨架，每个主题系列在此基础上叠加专属维度。

### 主骨架

| 层级 | 理论模型 | 核心问题 | 分析要点 |
|------|---------|---------|---------|
| **第一层：生命周期定位** | 企业生命周期 + 产品生命周期 | 这家公司和它的核心产品，现在在哪里？ | 企业所处阶段（初创/成长/成熟/衰退/重生）；产品所处阶段（导入/成长/成熟/衰退）；两者是否同步或错位 |
| **第二层：竞争格局** | 波特五力模型 | 护城河有多深？威胁从哪里来？ | 现有竞争者强度；供应商议价能力；买方议价能力；替代品威胁；新进入者威胁 |
| **第三层：财务质量验证** | 财务分析 | 战略选择在财务上得到了什么验证？ | 营收/利润趋势（≥3年）；毛利率/净利率/ROE；现金流质量；财务健康度 |
| **第四层：消费者洞察** | 消费者行为分析 | 消费者为什么选它，而不是对手？ | 目标用户画像；购买决策逻辑；情感驱动因素；生活方式关联 |

### 各系列专属叠加维度

| 主题系列 | 叠加维度 | 核心追问 |
|---------|---------|---------|
| 消费赛道50年陈系列 | 历史韧性维度 | 穿越了哪些危机？靠什么活下来？ |
| 消费出海企业系列 | 本土化策略维度 | 标准化输出 vs. 本土化适配，如何抉择？ |
| 新能源汽车消费系列 | 生活方式跃迁维度 | 改变了哪些出行习惯？消费者的购车决策逻辑是什么？ |
| 消费机器人系列 | 技术渗透曲线维度 | 现在处于S曲线哪个位置？消费者教育程度如何？ |
| 国潮2.0系列 | 品牌情感资产维度 | 民族情绪之外，产品力能否支撑溢价？ |
| AI重塑消费行业系列 | 技术重构维度 | AI改变了哪个环节？财务上体现在哪里？ |

## 五、内容格式规范

### 字数策略（分阶段执行）

| 粉丝阶段 | 目标字数 | 核心策略 |
|---------|---------|---------|
| **当前（<1万粉）** | **1200-1500字** | 培养阅读习惯，完读率优先于深度 |
| 成长期（1-3万粉） | 1500-2000字 | 读者已认可风格，逐步提升深度 |
| 成熟期（>3万粉） | 2000-3000字 | 深度长文为主，建立专业壁垒 |

### 层蛋糕结构（当前阶段标准结构）

```
【第一层：核心洞察层，600-800字】
  → 开篇（用具体场景/数据引入）
  → 核心观点（一个清晰的判断）
  → 关键论据（1-2个最有力的数据或案例）
  → 小结（读者在此可以停止，但已获得价值）

【第二层：深化分析层，400-500字】
  → 财务数据验证
  → 竞争格局延伸
  → 为想深入的读者补充背景

【第三层：收尾互动层，150-200字】
  → 一句话总结
  → 一个反问（引导留言）
  → 关注/加微信引导
```

## 六、文章类型与结构

### 类型1：企业深度研究（周内主力内容，每周4-5篇）

**文章目标**：用四层分析法 + 系列专属维度，拆解一家公司

**固定结构**：
```
标题：[企业名] + [核心观点/问题]（避免标题党，保持专业感）
开篇（150字）：具体场景或数据对比引入 → 提出核心问题
第一层：生命周期定位（200字）
第二层：竞争格局（300字）
第三层：财务质量（300字）
第四层：消费者洞察（300字）
结尾（150字）：一句话总结 → 抛出讨论话题 → 互动引导
```

### 类型2：行业观察（每周1篇，通常周末）

**文章目标**：跳出单一企业，看整个赛道的趋势与逻辑

**固定结构**：
```
标题：观点型 + 悬念
开篇（150字）：提出现象
第一部分：发生了什么（300字）
第二部分：为什么会这样（400字）
第三部分：未来会怎样（300字）
结尾（100字）：一句话总结
```

### 类型3：个人思考（每周1篇，通常周一或周日）

**文章目标**：建立人设，建立与读者的情感连接

**固定结构**：
```
标题：第一人称 + 真实感
开篇：一个具体场景或事件
正文：坦诚分享思考、困惑、选择
结尾：信念和行动
```
