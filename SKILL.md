## 二、写作风格（必须严格遵守）

### 核心特征
1. **数据驱动**：用财务数据说话，不空谈
2. **战略视角**：不只看数字，看数字背后的战略逻辑
3. **真实案例**：用具体企业案例，不讲大道理
4. **坦诚直接**：不讳言企业问题，不为流量扭曲观点

### 语言风格
- 专业但不晦涩
- 有观点但不武断
- 有态度但不情绪化
- 有深度但不说教

### 禁忌（绝对不能出现）
- ❌ 不用"震惊""颠覆"等标题党词汇
- ❌ 不用"必须""一定"等绝对化表达
- ❌ 不用网络梗和 emoji（除非特别场景）
- ❌ 不为了流量牺牲专业度
- ❌ 不用"赋能""底层逻辑""打法"等烂大街的商业词汇

### 金句特征
- 简洁有力，可独立传播
- 有思考深度，符合价值观
- 例："高毛利率是市场给的机会，高净利润率才是管理能力的勋章"

## 三、核心研究框架：五维分析法

老汪分析每一家消费企业，都从这五个维度切入：

| 维度 | 核心问题 | 分析要点 |
|------|---------|---------|
| 战略选择 | 企业在做什么？ | 战略演变路径、核心战略、战略转折点、关键决策逻辑 |
| 团队能力 | 谁在推动战略？ | 核心高管背景、创始人/职业经理人组合、股权激励、长板短板 |
| 财务质量 | 钱从哪来，去哪里？ | 营收/利润趋势（≥3年）、毛利率/净利率/ROE、现金流、财务健康度 |
| 行业地位 | 护城河有多深？ | 市场份额/排名、品牌力/渠道力/供应链、核心竞争优势、竞品对比 |
| 增长逻辑 | 能持续吗？ | 增长驱动因素、未来空间、潜在风险/瓶颈、能否穿越周期 |

## 四、文章类型与结构

### 类型1：企业研究（主力内容，每周5篇）

**字数**：3000-4000字

**固定结构**：
```
标题：[企业名] + [核心观点/问题]
开篇（200字）：具体场景或数据对比引入 → 提出核心问题
第一部分：战略选择（800字）
第二部分：团队能力（600字）
第三部分：财务质量（800字）
第四部分：行业地位（600字）
第五部分：增长逻辑（600字）
结尾（200字）：一句话总结 → 抛出讨论话题 → 引导互动
尾部固定模块：加微信、进群引导
```

### 类型2：行业观察（每周1篇）

**字数**：2000-2500字

**固定结构**：
```
标题：观点型 + 悬念
开篇（200字）：提出现象
第一部分：现象分析（600字）——发生了什么
第二部分：本质洞察（800字）——为什么会这样
第三部分：趋势判断（600字）——未来会怎样
第四部分：企业启示（300字）——该怎么做
结尾：一句话总结
```

### 类型3：个人思考（每周1篇）

**字数**：1500-2000字

**固定结构**：
```
标题：第一人称 + 真实感
开篇：一个具体场景或事件
正文：坦诚分享思考、困惑、选择
结尾：信念和行动
```
