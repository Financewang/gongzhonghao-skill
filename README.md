# 老汪的Financial笔记 — 公众号写作 Skill

> 消费企业战略观察 | 系统化创作工作台

## 这是什么？

这是"老汪的Financial笔记"公众号的**完整写作系统**，包含：定位、风格、研究框架、选题库、素材库、文稿模板和方法论。

## 核心理念

**从碎片化创作到系统化创作**：每次创作都在给系统添砖加瓦，而不是重新发明轮子。

## 如何与 AI 协作

1. 新对话时，发送本仓库链接
2. AI 读取 [SKILL.md](SKILL.md) 即可了解全部上下文
3. 使用指令（如"生成企业研究：伊利股份"）开始协作

## 仓库结构

| 文件夹 | 内容 |
|--------|------|
| `SKILL.md` | AI 协作总纲（最重要的文件） |
